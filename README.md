# shortshell

自己写的或收集的一些简短、有用的linux 脚本。不是写脚本多难，是哪怕自己写的，工作良好的，时间一长也就忘了。欢迎大家fork脚本。
请把脚本的功能写入到同名的不带扩展名的文件中。



| 文件夹      | 用途        | 备注                          |
| -------- | --------- | --------------------------- |
| lessfile | 用于减小文件的大小 |                             |
| init.d   | 一些有用的开机脚本 | (如果你不喜欢systemd或者使用openwrt等) |
